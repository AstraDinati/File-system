Функция rmdir используется для удаления папок. Пример:

<?php
	rmdir('dir');

нашёл вот такую рекурсивную функцию для удаления папки с файлами внутри в интернете:

$dir = 'link/to/your/dir';
function deleteDirectory($dir) {
    if (!file_exists($dir)) {
        return true;
    }

    if (!is_dir($dir)) {
        return unlink($dir);
    }

    foreach (scandir($dir) as $item) {
        if ($item == '.' || $item == '..') {
            continue;
        }

        if (!deleteDirectory($dir . DIRECTORY_SEPARATOR . $item)) {
            return false;
        }

    }

    return rmdir($dir);
}

3. Переименование и перемещение папок
Функция rename, которую вы уже знаете, может использоваться не только для работы с файлами, но и для папок.

Давайте для примера переименуем одну папку на другую:

<?php
	rename('old', 'new');
?>
А теперь выполним перемещение папки, одновременно переименовав ее:

<?php
	rename('old', 'dir/new');
?>

*) Пусть в корне вашего сайта лежит папка dir. Переименуйте ее на dir1.
